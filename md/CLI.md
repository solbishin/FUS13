# CLI 명령어

* pwd : 현재 디렉토리 확인하기
* cd :  디렉토리 이동하기
* mkdir : 디렉토리 생성하기
* touch : 내용이 들어있지 않는 빈 파일 생성
* echo : 내용이 들어있는 파일 생성
* mv : 파일, 디렉토리 이동 or 이름 변경
* cp : copy 복사하기
* rmdir : 디렉토리 제거하기(해당 폴더가 비어있어야 함)
* rm : 파일 제거하기
* && : 다중 명령어 수행 연결 접속사
* ls : 리스트 확인
* cat : 파일 내용 확인

```md
mkdir fus13(폴더명)
```
